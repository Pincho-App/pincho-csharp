# Code of Conduct

## Expected Behavior

- **Be kind and respectful** to others
- **Be constructive** with feedback and criticism
- **Assume good intentions** from others
- **Focus on what's best** for the project and community
- **Accept responsibility** when you make mistakes

## Unacceptable Behavior

Don't:
- Harass, insult, or demean anyone
- Use sexualized language or imagery
- Troll or make deliberately inflammatory comments
- Publish private information without consent
- Act unprofessionally or disrespectfully

## Reporting Issues

If you experience or witness unacceptable behavior, report it to **support@pincho.app**.

All reports will be reviewed and handled with discretion. We'll keep your information confidential.

## Enforcement

Violations may result in:

1. **Warning** - Private communication about the issue
2. **Temporary restriction** - Time away from the project
3. **Permanent ban** - Removal from the community

We'll handle each situation fairly and consider context, but we take violations seriously.

## Scope

This code of conduct applies to all project spaces (issues, pull requests, discussions) and when representing the project publicly (social media, events, etc.).
